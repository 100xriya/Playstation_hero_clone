PlayStation Homepage Clone

This is a simple front-end clone of the PlayStation homepage built using HTML and CSS. The project focuses on recreating the navigation bar and the GTA VI hero section while practicing modern CSS layout and styling techniques.

Features
Navigation bar similar to the PlayStation website
GTA VI hero section
Gradient overlay on the background image
Hover effects on buttons
Flexbox layout
Built With
HTML5
CSS3
Flexbox
What I Learned
Creating hero sections with background-image
Using linear-gradient() to improve text readability
The difference between using <img> and background-image
Building layouts with Flexbox
Debugging CSS using browser DevTools
Challenges

Some of the things I found challenging while building this project were:

Understanding how gradient overlays work
Positioning text over a background image
Fixing CSS rules that were overriding each other
Choosing when to use <img> versus background-image
